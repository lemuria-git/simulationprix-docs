# Référentiels (`ref.*`)

Les référentiels structurent la base. Ils représentent les **axes de calcul**, les **segments clients** et les **unités de durée**.

## ref.DureeTypeProduit
Décrit les durées standard par type de produit :  
4h, 1j, 2j, 3j, 4j, 5j, 6j... jusqu'à 21j.

## ref.Categorie
Liste les catégories clients :  
Adulte, Enfant, Senior, Étudiant…

## ref.ConditionCo
Décrit les conditions commerciales :  
Public, Groupe, Internet, CE…

## ref.FenetreAchat
Découpage temporel :  
Premières neiges, Pleine saison, Fin de saison...

## ref.GroupePool / ref.Pool
Définit les périmètres commerciaux et techniques.

---
**Idée clé :**  
Les référentiels donnent la « géométrie » du modèle.
