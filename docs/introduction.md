# Introduction

La plateforme **SimulationPrix** est conçue pour permettre la création, la comparaison et la compréhension de simulations tarifaires pour des produits complexes (forfaits, billets, durées multiples, catégories clients, fenêtres d’achat, etc.).

Cette documentation explique :

- la logique globale du système,
- la structure des données,
- le fonctionnement du calcul,
- le rôle des outils (Power Apps, SQL, Power BI),
- les bonnes pratiques pour maintenir une cohérence globale.

Elle est volontairement rédigée en **français courant**, accessible et pédagogique.
