# Power BI

Power BI consomme les vues :

- `pbi.vw_ResultatPrix`
- `pbi.vw_Comparaison`
- `pbi.vw_ResultatsStandardises`

## Usages

- comparaison inter-simulations,
- analyse par durée, fenêtre, catégorie,
- suivi de la stratégie tarifaire,
- export et reporting.

## Recommandation
Utiliser DirectQuery pour des données toujours fraîches.
