# Configuration (`cfg.*`)

La configuration définit les règles versionnées.

## cfg.VersionRegles
Chaque version représente un jeu de barèmes figé.

## cfg.RemiseCategorie
Remises par catégorie (ex : Enfant -20 %, Senior -10 %).

## cfg.RemiseConditionCo
Remises par condition commerciale (ex : Groupe -15 %).

## cfg.FenetreAchatExercice
Association dynamique entre fenêtres d’achat et Exercice/Période/Périmètre.

---
**À retenir :**  
La configuration est la couche « vivante » du système.
